Nosotros

Somos un equipo con una pasión por la tecnología

CoDev es un equipo de profesionales involucrados en el mundo de la tecnología. Nuestro compromiso es brindar a nuestros clientes un excelente servicio en la implementación del desarrollo de software, diseño gráfico y web, posicionamiento SEO y marketing digital.

Misión

Ser la propuesta económica, creativa y de cumplimiento a corto plazo para el desarrollo de proyectos de software, a personas naturales, pymes o cualquier otra entidad o parte interesada

Visión

Contribuir con el alcance de los objetivos y metas de nuestros clientes mediante la implementación de nuestros servicios, basados en el diseño y desarollo de proyectos de software

Valores

Somos un equipo caracterizado por el compromiso, honestidad, creatividad, empeño, y entrega en tiempo y forma en el diseño y desarrollo de los proyectos de software de nuestros clientes

Tecnologías

Diseñamos y desarrollamos tus proyectos con las actuales tecnologías del mercado

html5

Es un lenguaje de marcado descriptivo que se escribe en forma de etiquetas para definir la estructura de una página web

css3

Es un lenguaje utilizado para organizar la presentación y aspectos de los documentos html

javascript

Es un lenguaje interpretado que se utiliza principalmente en el lado del cliente, permitiendo crear páginas web dinámicas

pug

Es un motor de plantillas, basado en JavaScript, para la compilación resultante de código html

sass

Es un metalenguaje de hojas de estilo en cascada. Es un lenguaje de script que es traducido a css

npm

Es el administrador de paquetes para JavaScript y el registro de software más grande del mundo

java

Es un lenguaje de programación de propósito general, concurrente, orientado a objetos

postgresql

Es un sistema de gestión de bases de datos relacional orientado a objetos y libre, publicado bajo la licencia PostgreSQL

sql server

Es un sistema de manejo de bases de datos del modelo relacional, desarrollado por la empresa Microsoft

git

Es un sistema de control de versiones, que nos proporciona herramientas para desarrollar un trabajo en equipo de manera inteligente y rápida

gulp

Es un sistema de construcción que permite automatizar tareas, tales como minificar y validar código JavaScript, CSS y HTML

maven

Es una herramienta de software para la gestión y construcción de proyectos Java

adobe illustrator

Es un editor de gráficos vectoriales en forma de taller de arte, destinado a la creación artística de dibujo y pintura para ilustración

adobe xd

Es una aplicación de software de diseño de la experiencia del usuario que admite el diseño vectorial, wireframing y prototipado
